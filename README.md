This project is a Node.js-based web platform that integrates multiple external APIs to generate a random user profile and provide contextual data based on that user’s origin.
Random User Creator: Retrieves a random profile featuring name, gender, age, date of birth, and location.
REST Countries API: It fetches the capital city, official languages, currency, and national flag based on the user's country.
Exchange Rate API: Offers a real-time assessment of the user's native currency in relation to the United States Dollar (USD) and Kazakhstani Tenge (KZT).
News API: Retrieves five latest headlines in English that reference the user's country, along with images and source links
API:
Random User Generator API - https://randomuser.me/api/
Integrate REST Countries API based on user’s country - https://restcountries.com/#endpoints
Add Exchange Rate API based on user’s currency - https://www.exchangerate-api.com/
Show headlines with News API based on the user's country - https://newsapi.org/
